# Student Grade Calculator

## Project Overview
This project is a beginner-level Python application that calculates student grades based on marks entered by the user.

## Objectives
- Learn if-elif-else statements
- Understand loops and input validation
- Create and use functions
- Build an interactive Python application

## Technologies Used
- Python 3
- Visual Studio Code

## Grading Logic
A : 90-100
B : 80-89
C : 70-79
D : 60-69
F : 0-59

## Features
- Takes student name and marks
- Validates marks between 0 and 100
- Calculates grades automatically
- Displays encouraging messages

## How to Run

python grade_calculator.py

## What I Learned
- If-elif-else statements
- While loops
- Functions
- Input validation
- Writing modular programs
